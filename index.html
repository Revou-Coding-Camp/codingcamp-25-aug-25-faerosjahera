My Project

Aplikasi Pemesanan Gacoan

Deskripsi: Aplikasi ini dirancang sebagai layanan pesan antar makanan berbasis mobile untuk produk Gacoan. Dikembangkan menggunakan Dart (Flutter) untuk frontend, PHP untuk backend, dan Postman untuk pengujian serta integrasi API. Untuk perancangan antarmuka dilakukan dengan Figma

Tantangan: Tidak ada sistem pembayaran asli atau data transaksi riil yang diintegrasikan, sehingga pengujian dilakukan secara simulasi.

Solusi: Menggunakan Postman untuk menguji setiap endpoint API sebelum diintegrasikan ke aplikasi, sehingga dapat meminimalisir bug saat penggabungan frontend dan backend.

Hasil: Aplikasi berhasil dijalankan secara lokal dengan fitur utama seperti pemilihan menu, keranjang belanja, dan simulasi konfirmasi pesanan. Aplikasi juga berhasil dijalankan pada perangkat Android melalui proses build dan deployment langsung (APK), meskipun belum dirilis di Play Store.

Website Pendeteksi Penyakit Asma "BreathEase"

Deskripsi: Situs web sistem pakar yang dirancang untuk membantu pengguna dalam mendeteksi gejala asma berdasarkan input gejala yang dirasakan. Sistem ini dikembangkan menggunakan Flask sebagai backend dan tools frontend seperti HTML, CSS, dan JavaScript, dengan desain antarmuka yang dibuat agar mudah diakses oleh pengguna awam.

Tantangan:Dalam pengembangan sistem pakar, idealnya diperlukan konsultasi langsung dengan dokter atau pakar kesehatan untuk merancang logika diagnosis yang akurat.

Solusi: Mengatasi keterbatasan tersebut dengan menggunakan referensi dari sumber medis tepercaya yaitu dari WHO.

Hasil: Sistem mampu memberikan hasil deteksi gejala asma yang logis berdasarkan input pengguna.
